# 🎓 Placement Analyzer Web Application

## 📌 Project Overview

The Placement Analyzer is a web-based application designed to evaluate a student's qualifications, skills, and academic performance to determine their eligibility for internship placements.

This system helps students understand their readiness for internships by providing:

* Eligibility status
* Performance score
* Improvement suggestions

It is developed as a **mini project** using basic web technologies.

---

## 🚀 Features

✅ Student qualification analysis
✅ CGPA evaluation (supports decimal values like 5.67, 8.25)
✅ Skills assessment
✅ Experience evaluation
✅ Internship eligibility prediction
✅ Score calculation out of 100
✅ Personalized improvement suggestions
✅ Simple and user-friendly interface

---

## ⚙️ How It Works

1. User enters details:

   * Name
   * CGPA
   * Qualification
   * Skills
   * Experience

2. The system analyzes the data using predefined logic.

3. A score is calculated based on:

   * Academic performance
   * Technical skills
   * Qualification level
   * Project experience

4. The result is displayed with:

   * Score
   * Eligibility status
   * Suggestions for improvement

---

## 📊 Scoring Criteria

| Criteria      | Marks |
| ------------- | ----- |
| CGPA          | 30    |
| Qualification | 20    |
| Skills        | 30    |
| Experience    | 20    |
| Total         | 100   |

---

## ✅ Eligibility Logic

* **Score ≥ 70** → Eligible for Internship
* **Score 40 – 69** → Needs Improvement
* **Score < 40** → Not Eligible

---

## ▶️ How to Run the Project

1. Download or clone the project folder.
2. Open `index.html` in any web browser.
3. Enter student details.
4. Click **Analyze** to view results.

No installation required.
